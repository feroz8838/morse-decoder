# 🌐 Morse Code Encoder & Decoder

This repository contains simple programs to **encode text into Morse code** and **decode Morse code back into text**.

The goal of this project is not just functionality, but to build a **multi-language collection** of Morse code tools.

---

## 🚀 Features

* Encode text → Morse code
* Decode Morse code → text
* Supports:

  * Alphabets (A–Z)
  * Numbers (0–9)
  * Common punctuation
* Beginner-friendly implementation

---

## 🧑‍💻 Languages Supported

Currently implemented in:

* C
* Python

👉 More languages are welcome!

## 📌 Contribution Guidelines

1. Fork the repository
2. Create a new folder for your language (e.g., `java/`, `rust/`, `go/`)
3. Add your encoder/decoder implementation
4. Keep the code simple and readable
5. Submit a pull request
